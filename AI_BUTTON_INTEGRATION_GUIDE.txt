================================================================================
  COMMON ISSUES & FIXES
================================================================================

ISSUE: "VITE_ANTHROPIC_API_KEY not configured"
FIX: Make sure .env file has VITE_ANTHROPIC_API_KEY=sk-ant-...

ISSUE: "Skin_Care.csv not found"
FIX: Verify Skin_Care.csv exists in glow12.0 root directory

ISSUE: "Could not import skin_analysis_endpoint.py"
FIX: Make sure skin_analysis_endpoint.py is in the same folder as server.py

ISSUE: Claude API returns 401 error
FIX: Check that API key is valid and not expired

ISSUE: Image upload fails
FIX: Try with JPG/PNG image, clear browser cache, check file size < 5MB
