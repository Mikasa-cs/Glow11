🆘 QUICK TROUBLESHOOTING

    Backend won't start?
    → Make sure Python is installed: python --version
    → Run: python server.py

    Frontend won't start?
    → Make sure Node.js is installed: npm --version
    → Run: npm run dev

    Login fails?
    → Email is case-insensitive, password is case-sensitive
    → Try: shivi5035singh@gmail.com / QWERTY@123

    Admin registration fails?
    → Check existing admin credentials are correct
    → New password must be at least 6 characters
    → New email must not already exist
